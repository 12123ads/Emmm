# Emmm
可能是最简单的服务器错误页面

## 使用方法
1. `Star` 本项目
2. 从这里 [下载](https://github.com/Dreamer-Paul/Emmm/archive/master.zip) 主题源码
3. 修改模板的内容与链接
4. 上传到你的服务器即可食用~
